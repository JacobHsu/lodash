

## [isArray](https://www.html.cn/doc/lodash/#_isarrayvalue)

> _.isArray(value)

检查 value 是否是 Array 类对象。

```js
_.isArray([1, 2, 3]);
// => true
 
_.isArray(document.body.children);
// => false
 
_.isArray('abc');
// => false
```

## [isString](https://www.html.cn/doc/lodash/#_isstringvalue)

> _.isString(value)

检查 value 是否是原始字符串String或者对象。

```js
_.isString('abc');
// => true
 
_.isString(1);
// => false
```
